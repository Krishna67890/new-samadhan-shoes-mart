NEW SAMADHAN SHOES MART - INSTALLATION GUIDE

Follow these steps to set up the project locally:

1. PREREQUISITES:
   - Node.js (v16 or higher)
   - npm (v8 or higher)
   - A modern web browser (Chrome/Edge recommended for Speech API)

2. CLONE & SETUP:
   - Extract the project folder.
   - Open your terminal/command prompt.
   - Navigate to the project root: `cd "New Samadhan Shoes"`

3. FRONTEND INSTALLATION:
   - Navigate to the frontend directory: `cd frontend`
   - Install dependencies: `npm install`
   - If you encounter peer dependency issues, use: `npm install --legacy-peer-deps`

4. RUNNING THE APPLICATION:
   - While in the `frontend` folder, start the development server: `npm start`
   - The website will open at: `http://localhost:3000`

5. PRODUCTION BUILD:
   - To create a production-ready build: `npm run build`
   - The output will be in the `frontend/build` folder.

NOTE: This is a "Local-First" application. It does not require a running backend/database for basic identity and ordering functionality as it uses the browser's LocalStorage and direct WhatsApp integration.

# New Samadhan Shoes Mart - Full System Recovery

A premium, local-first boutique footwear web application designed for elite performance and seamless direct-to-shopkeeper ordering.

## 🚀 Key Features

- **Local-First Identity Sync**: No traditional registration. User data is stored securely in `localStorage` for instant, friction-free access.
- **Dual-Direct WhatsApp Protocol**: Orders are automatically formatted and sent directly to two official shopkeeper numbers for immediate processing.
- **Elite AI Voice Assistant**: Context-aware voice guidance (en-IN) providing product insights and navigation help across the store.
- **Zero-Blur UI Standard**: High-contrast, accessibility-focused design with a strict no-backdrop-filter policy for maximum legibility.
- **Nashik Localization**: Integrated business logic specifically tailored for the Nashik, Maharashtra region.

## 🛠️ Tech Stack

- **Frontend**: React.js, Tailwind CSS
- **Animations**: GSAP (GreenSock Animation Platform)
- **Icons**: Lucide React
- **Voice**: Web Speech API (SpeechSynthesis)
- **State Management**: Context API & LocalStorage

## 📦 Project Structure

- `frontend/`: React application and assets.
- `backend/`: (Legacy) Residual MongoDB/Express logic, now bypassed by the Local-First architecture.
